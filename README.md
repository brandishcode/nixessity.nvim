# Nixessity
For all your nix flake command necessities.

> [!WARNING]
> Plugin is still in development.

# Setup
```lua
require"nixessity".setup()
```

# Features
|commands|arguments|definitions|
|-|-|-|
|Nixhelp|targetCmd {string}|Print the target nix command documentation|
